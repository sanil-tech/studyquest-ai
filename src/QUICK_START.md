# 🚀 Quick Start Guide - Create Student Account

## Step 1: Create Parent Account
1. Go to `/register`
2. Fill in:
   - Email: (your email)
   - Password: (your password)
   - Confirm Password
3. Click "Register"
4. Select role: **"Parent"**
5. Complete profile:
   - Full Name
   - Phone Number
   - Number of Children
6. Click "Continue" → You're now in Parent Dashboard!

## Step 2: Create Child Account
1. In Parent Dashboard, click **"Add Child"** button
2. Fill in child details:
   - **Full Name**: (required - must be real name, not empty!)
   - **Date of Birth**: (e.g., 2015-01-01)
   - **Gender**: Male/Female
   - **State**: (e.g., Kuala Lumpur)
   - **Profile Photo**: (optional)
3. Click **"Create Account"**
4. **IMPORTANT**: You'll see credentials displayed:
   - **Student ID**: (e.g., SQ-8F3K92) ← Write this down!
   - **Password**: (e.g., aB3xK9mN) ← Write this down!
   - **PIN**: (if enabled)

## Step 3: Student Login
1. Logout from parent account (or open new incognito window)
2. Go to `/login`
3. Click **"Child Login"** button (with graduation cap icon)
4. Enter:
   - **Student ID**: (from Step 2)
   - **Password**: (from Step 2)
5. Click "Login 🎯"
6. Complete profile if prompted
7. Welcome to Student Dashboard! 🎉

---

## ⚠️ Common Issues

### "Incorrect Student ID"
- **Cause**: No child account exists yet
- **Fix**: Follow Step 1 & 2 above to create child account

### "Profile Incomplete"
- **Cause**: Child account created but profile not finished
- **Fix**: Complete the profile wizard after login

### "Incorrect details"
- **Cause**: Wrong password or Student ID
- **Fix**: 
  - Check spelling (Student ID is uppercase: SQ-XXXXXX)
  - Parent can reset credentials in Child Profile page

### Can't find "Add Child" button
- **Cause**: Not logged in as parent
- **Fix**: Login with parent account first
